# Webpack-template

A simple and interactive **ToDo Application** built with **Vanilla JavaScript** and bundled using **Webpack**.  
This app allows users to create, edit, delete, and manage tasks efficiently with project-based organization.

## Installation

> **Note:** The `node_modules` folder is not included in the repository.
 Run `npm install` to install all dependencies before running the project.


## Development
npm run start

## Build
npm run build
